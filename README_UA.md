## Огляд

**Datvex Prompt LAB** — це професійний структурований репозиторій, що містить високоякісні промпти, розроблені для спеціалізованих завдань у різних системах ШІ. Платформа надає інтуїтивний інтерфейс з темною темою для перегляду, пошуку та фільтрації промптів за типом, категорією та тегами.

Створений з фокусом на продуктивність, Datvex забезпечує блискавичний досвід використання з миттєвим пошуком, нескінченною прокруткою та потужною системою фільтрації — все без важких фреймворків.

## Внесок

Внески — це те, що робить спільноту відкритого коду чудовою. Будь-який ваш внесок **дуже цінується**.

### Як Зробити Внесок

1. **Зробіть Fork** репозиторію
2. **Створіть** гілку для вашої функції
   ```bash
   git checkout -b feature/чудова-функція
   ```
3. **Зафіксуйте** ваші зміни
   ```bash
   git commit -m "Додає чудову функцію"
   ```
4. **Відправте** в гілку
   ```bash
   git push origin feature/чудова-функція
   ```
5. **Відкрийте** Pull Request

### Стиль Коду

- У коді уникайте надмірної кількості коментарів — допускаються лише суттєві коментарі або коментарі в особливо складних місцях
- Використовуйте утилітарні класи TailwindCSS для стилізації
- Підтримуйте однорідність темної теми
- Використовуйте чистий JavaScript (без фреймворків)
