## Panoramica

**Datvex Prompt LAB** è un repository professionale e strutturato contenente prompt di alta qualità progettati per compiti specializzati su molteplici sistemi di IA. La piattaforma offre un'interfaccia intuitiva con tema scuro per navigare, cercare e filtrare i prompt per tipo, categoria e tag.

Costruito con le prestazioni in mente, Datvex offre un'esperienza ultra-veloce con ricerca istantanea, scroll infinito e un potente sistema di filtraggio — il tutto senza framework pesanti.

## Contributi

I contributi sono ciò che rende la comunità open source straordinaria. Qualsiasi contributo è **molto apprezzato**.

### Come Contribuire

1. **Forka** il repository
2. **Crea** il tuo branch di funzionalità
   ```bash
   git checkout -b feature/funzionalita-incredibile
   ```
3. **Committa** le tue modifiche
   ```bash
   git commit -m "Aggiunge funzionalità incredibile"
   ```
4. **Pusha** verso il branch
   ```bash
   git push origin feature/funzionalita-incredibile
   ```
5. **Apri** una Pull Request

### Stile del Codice

- Nel codice, evita un eccesso di commenti — sono ammessi solo commenti essenziali o in aree particolarmente complesse
- Usa le classi utility di TailwindCSS per lo stile
- Mantieni la coerenza del tema scuro
- Mantieni JavaScript vanilla (senza framework)
