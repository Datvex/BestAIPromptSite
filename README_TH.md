## ภาพรวม

**Datvex Prompt LAB** เป็นคลังข้อมูลที่มีโครงสร้างอย่างมืออาชีพ ประกอบด้วยพรอมต์คุณภาพสูงที่ออกแบบมาสำหรับงานเฉพาะทางในระบบ AI หลากหลาย แพลตฟอร์มนี้มีอินเทอร์เฟซธีมมืดที่ใช้งานง่ายสำหรับการเรียกดู ค้นหา และกรองพรอมต์ตามประเภท หมวดหมู่ และแท็ก

สร้างขึ้นโดยคำนึงถึงประสิทธิภาพ Datvex มอบประสบการณ์ที่รวดเร็วดุจสายฟ้าด้วยการค้นหาแบบทันที การเลื่อนแบบไม่มีที่สิ้นสุด และระบบกรองที่ทรงพลัง — ทั้งหมดนี้โดยไม่ต้องใช้เฟรมเวิร์กที่หนัก

## การมีส่วนร่วม

การมีส่วนร่วมคือสิ่งที่ทำให้ชุมชนโอเพนซอร์สยอดเยี่ยม การมีส่วนร่วมใดๆ ของคุณจะ**ได้รับการชื่นชมอย่างยิ่ง**

### วิธีการมีส่วนร่วม

1. **Fork** คลังข้อมูล
2. **สร้าง** แบรนช์ฟีเจอร์ของคุณ
   ```bash
   git checkout -b feature/amazing-feature
   ```
3. **Commit** การเปลี่ยนแปลงของคุณ
   ```bash
   git commit -m "เพิ่มฟีเจอร์ที่ยอดเยี่ยม"
   ```
4. **Push** ไปยังแบรนช์
   ```bash
   git push origin feature/amazing-feature
   ```
5. **เปิด** Pull Request

### สไตล์โค้ด

- ในโค้ด หลีกเลี่ยงการมีคอมเมนต์มากเกินไป — อนุญาตเฉพาะคอมเมนต์ที่จำเป็นหรือในส่วนที่ซับซ้อนเป็นพิเศษเท่านั้น
- ใช้ TailwindCSS utility classes สำหรับการจัดสไตล์
- รักษาความสอดคล้องของธีมมืด
- ใช้ JavaScript แบบ vanilla (ไม่ใช้เฟรมเวิร์ก)
