## 개요

**Datvex Prompt LAB**은 여러 AI 시스템의 전문 작업을 위해 설계된 고품질 프롬프트를 포함하는 전문적이고 체계적인 저장소입니다. 이 플랫폼은 유형, 카테고리 및 태그별로 프롬프트를 탐색, 검색 및 필터링할 수 있는 직관적인 다크 테마 인터페이스를 제공합니다.

성능을 최우선으로 구축된 Datvex는 즉각적인 검색, 무한 스크롤, 강력한 필터링 시스템을 통해 번개처럼 빠른 경험을 제공합니다 — 무거운 프레임워크 없이 모두 구현됩니다.

## 기여하기

기여는 오픈소스 커뮤니티를 놀라운 곳으로 만드는 원동력입니다. 여러분의 모든 기여는 **매우 감사히 여깁니다**.

### 기여 방법

1. 저장소를 **Fork** 합니다
2. 기능 브랜치를 **생성** 합니다
   ```bash
   git checkout -b feature/amazing-feature
   ```
3. 변경사항을 **커밋** 합니다
   ```bash
   git commit -m "놀라운 기능 추가"
   ```
4. 브랜치에 **푸시** 합니다
   ```bash
   git push origin feature/amazing-feature
   ```
5. Pull Request를 **오픈** 합니다

### 코드 스타일

- 코드에서 과도한 주석을 피하세요 — 필수적인 주석이나 특히 복잡한 영역의 주석만 허용됩니다
- 스타일링에는 TailwindCSS 유틸리티 클래스를 사용하세요
- 다크 테마의 일관성을 유지하세요
- JavaScript는 바닐라를 유지하세요 (프레임워크 없음)
