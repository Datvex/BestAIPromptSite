## Oversikt

**Datvex Prompt LAB** er et profesjonelt, strukturert repositorium som inneholder høykvalitets-prompter designet for spesialiserte oppgaver på tvers av flere AI-systemer. Plattformen tilbyr et intuitivt grensesnitt med mørkt tema for å bla gjennom, søke og filtrere prompter etter type, kategori og tagger.

Bygget med ytelse i tankene, leverer Datvex en lynrask opplevelse med øyeblikkelig søk, uendelig scrolling og et kraftig filtreringssystem — alt uten tunge rammeverk.

## Bidra

Bidrag er det som gjør åpen kildekode-fellesskapet fantastisk. Ethvert bidrag du gir, er **svært verdsatt**.

### Hvordan Bidra

1. **Fork** repositoriet
2. **Opprett** din funksjonsgren
   ```bash
   git checkout -b feature/fantastisk-funksjon
   ```
3. **Commit** endringene dine
   ```bash
   git commit -m "Legg til fantastisk funksjon"
   ```
4. **Push** til grenen
   ```bash
   git push origin feature/fantastisk-funksjon
   ```
5. **Åpne** en Pull Request

### Kodestil

- I koden, unngå overflødige kommentarer — bare essensielle kommentarer eller de i spesielt komplekse områder er tillatt
- Bruk TailwindCSS utility-klasser for styling
- Oppretthold det mørke temaets konsistens
- Hold JavaScript vanilla (ingen rammeverk)
