## Descripción General

**Datvex Prompt LAB** es un repositorio profesional y estructurado que contiene prompts de alta calidad diseñados para tareas especializadas en múltiples sistemas de IA. La plataforma proporciona una interfaz intuitiva con tema oscuro para navegar, buscar y filtrar prompts por tipo, categoría y etiquetas.

Construido pensando en el rendimiento, Datvex ofrece una experiencia ultrarrápida con búsqueda instantánea, desplazamiento infinito y un potente sistema de filtros — todo sin frameworks pesados.

## Contribuir

Las contribuciones son lo que hace increíble a la comunidad de código abierto. Cualquier contribución que hagas es **muy apreciada**.

### Cómo Contribuir

1. **Haz un Fork** del repositorio
2. **Crea** tu rama de funcionalidad
   ```bash
   git checkout -b feature/nueva-funcionalidad
   ```
3. **Haz commit** de tus cambios
   ```bash
   git commit -m "Agregar nueva funcionalidad"
   ```
4. **Envía** a la rama
   ```bash
   git push origin feature/nueva-funcionalidad
   ```
5. **Abre** un Pull Request

### Estilo de Código

- En el código, evita el exceso de comentarios — solo se permiten comentarios esenciales o los de áreas particularmente complejas
- Usa clases utilitarias de TailwindCSS para estilizar
- Mantén la consistencia del tema oscuro
- Mantén JavaScript puro (sin frameworks)
