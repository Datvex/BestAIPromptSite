## Pangkalahatang-ideya

Ang **Datvex Prompt LAB** ay isang propesyonal, nakabalangkas na repository na naglalaman ng mataas na kalidad na mga prompt na dinisenyo para sa mga espesyalisadong gawain sa maraming AI system. Ang platform ay nagbibigay ng isang intuitive na dark-themed na interface para sa pag-browse, paghahanap, at pag-filter ng mga prompt ayon sa uri, kategorya, at mga tag.

Binuo na may pagganap sa isip, ang Datvex ay naghahatid ng mabilis-na-parang-kidlat na karanasan na may instant na paghahanap, walang-katapusang pag-scroll, at isang makapangyarihang sistema ng pag-filter — lahat nang walang mabibigat na framework.

## Pag-ambag

Ang mga kontribusyon ang siyang gumagawa sa open source na komunidad na kamangha-mangha. Anumang kontribusyon mo ay **lubos na pinahahalagahan**.

### Paano Mag-ambag

1. **I-Fork** ang repository
2. **Gumawa** ng iyong feature branch
   ```bash
   git checkout -b feature/kahanga-hangang-tampok
   ```
3. **I-Commit** ang iyong mga pagbabago
   ```bash
   git commit -m "Magdagdag ng kahanga-hangang tampok"
   ```
4. **I-Push** sa branch
   ```bash
   git push origin feature/kahanga-hangang-tampok
   ```
5. **Magbukas** ng Pull Request

### Estilo ng Code

- Sa code, iwasan ang labis na mga komento — ang mahahalagang komento lamang o ang nasa partikular na kumplikadong mga lugar ang pinapayagan
- Gumamit ng TailwindCSS utility classes para sa styling
- Panatilihin ang pagkakapare-pareho ng dark theme
- Panatilihin ang JavaScript na vanilla (walang mga framework)
