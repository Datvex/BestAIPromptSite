## Tổng Quan

**Datvex Prompt LAB** là một kho lưu trữ chuyên nghiệp, có cấu trúc chứa các prompt chất lượng cao được thiết kế cho các tác vụ chuyên biệt trên nhiều hệ thống AI. Nền tảng cung cấp giao diện trực quan với giao diện tối để duyệt, tìm kiếm và lọc prompt theo loại, danh mục và thẻ.

Được xây dựng với hiệu suất là ưu tiên hàng đầu, Datvex mang đến trải nghiệm nhanh như chớp với tìm kiếm tức thì, cuộn vô hạn và hệ thống lọc mạnh mẽ — tất cả đều không cần framework nặng.

## Đóng Góp

Đóng góp là điều làm cho cộng đồng mã nguồn mở trở nên tuyệt vời. Mọi đóng góp của bạn đều được **đánh giá cao**.

### Cách Đóng Góp

1. **Fork** kho lưu trữ
2. **Tạo** nhánh tính năng của bạn
   ```bash
   git checkout -b feature/tinh-nang-tuyet-voi
   ```
3. **Commit** các thay đổi của bạn
   ```bash
   git commit -m "Thêm tính năng tuyệt vời"
   ```
4. **Push** lên nhánh
   ```bash
   git push origin feature/tinh-nang-tuyet-voi
   ```
5. **Mở** một Pull Request

### Phong Cách Code

- Trong code, tránh thừa comment — chỉ cho phép comment cần thiết hoặc ở những vùng đặc biệt phức tạp
- Sử dụng các lớp tiện ích TailwindCSS để tạo kiểu
- Duy trì tính nhất quán của giao diện tối
- Giữ JavaScript thuần (không dùng framework)
