## 概要

**Datvex Prompt LAB** は、複数のAIシステムにおける専門的なタスク向けに設計された高品質なプロンプトを含む、プロフェッショナルな構造化リポジトリです。プラットフォームは、タイプ、カテゴリ、タグでプロンプトを閲覧、検索、フィルタリングするための直感的なダークテーマインターフェースを提供します。

パフォーマンスを念頭に構築されたDatvexは、インスタント検索、無限スクロール、強力なフィルタリングシステムにより、超高速体験を提供します — すべて重いフレームワークなしで実現しています。

## コントリビューション

コントリビューションこそがオープンソースコミュニティを素晴らしいものにしています。あなたのいかなる貢献も**大変感謝いたします**。

### コントリビューション方法

1. リポジトリを**Fork**する
2. フィーチャーブランチを**作成**する
   ```bash
   git checkout -b feature/amazing-feature
   ```
3. 変更を**コミット**する
   ```bash
   git commit -m "素晴らしい機能を追加"
   ```
4. ブランチに**プッシュ**する
   ```bash
   git push origin feature/amazing-feature
   ```
5. Pull Requestを**オープン**する

### コードスタイル

- コード内では過剰なコメントを避ける — 必要不可欠なコメントや特に複雑な箇所のみ許可
- スタイリングにはTailwindCSSユーティリティクラスを使用
- ダークテーマの一貫性を維持
- JavaScriptはバニラを維持（フレームワーク不使用）
