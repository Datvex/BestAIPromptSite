## Преглед

**Datvex Prompt LAB** е професионално, структурирано хранилище, съдържащо висококачествени промпти, проектирани за специализирани задачи в множество AI системи. Платформата предоставя интуитивен интерфейс с тъмна тема за преглеждане, търсене и филтриране на промпти по тип, категория и тагове.

Изграден с фокус върху производителността, Datvex предоставя светкавично бързо изживяване с мигновено търсене, безкрайно превъртане и мощна система за филтриране — всичко без тежки фреймуърки.

## Принос

Приносите са това, което прави общността с отворен код невероятна. Всеки ваш принос е **високо оценен**.

### Как да Допринесете

1. **Направете Fork** на хранилището
2. **Създайте** вашия feature клон
   ```bash
   git checkout -b feature/страхотна-функция
   ```
3. **Направете Commit** на промените си
   ```bash
   git commit -m "Добавя страхотна функция"
   ```
4. **Направете Push** към клона
   ```bash
   git push origin feature/страхотна-функция
   ```
5. **Отворете** Pull Request

### Стил на Кода

- В кода избягвайте излишни коментари — допускат се само съществени коментари или такива в особено сложни области
- Използвайте TailwindCSS утилити класове за стилизиране
- Поддържайте консистентност на тъмната тема
- Запазвайте JavaScript ванилен (без фреймуърки)
