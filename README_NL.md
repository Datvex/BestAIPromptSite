## Overzicht

**Datvex Prompt LAB** is een professionele, gestructureerde repository met hoogwaardige prompts ontworpen voor gespecialiseerde taken over meerdere AI-systemen. Het platform biedt een intuïtieve interface met donker thema voor het bladeren, zoeken en filteren van prompts op type, categorie en tags.

Gebouwd met prestaties in gedachten, levert Datvex een razendsnelle ervaring met directe zoekfunctie, oneindig scrollen en een krachtig filtersysteem — alles zonder zware frameworks.

## Bijdragen

Bijdragen zijn wat de open source community geweldig maken. Elke bijdrage die u levert, wordt **zeer gewaardeerd**.

### Hoe Bij te Dragen

1. **Fork** de repository
2. **Maak** uw feature-branch aan
   ```bash
   git checkout -b feature/geweldige-functie
   ```
3. **Commit** uw wijzigingen
   ```bash
   git commit -m "Voeg geweldige functie toe"
   ```
4. **Push** naar de branch
   ```bash
   git push origin feature/geweldige-functie
   ```
5. **Open** een Pull Request

### Codestijl

- Vermijd in de code een overvloed aan commentaar — alleen essentieel commentaar of commentaar in bijzonder complexe gebieden is toegestaan
- Gebruik TailwindCSS utility-klassen voor styling
- Behoud de consistentie van het donkere thema
- Houd JavaScript vanilla (geen frameworks)
