## Przegląd

**Datvex Prompt LAB** to profesjonalne, uporządkowane repozytorium zawierające wysokiej jakości prompty zaprojektowane do specjalistycznych zadań w wielu systemach AI. Platforma udostępnia intuicyjny interfejs z ciemnym motywem do przeglądania, wyszukiwania i filtrowania promptów według typu, kategorii i tagów.

Zbudowany z myślą o wydajności, Datvex zapewnia błyskawiczne doświadczenie z natychmiastowym wyszukiwaniem, nieskończonym przewijaniem i potężnym systemem filtrowania — wszystko bez ciężkich frameworków.

## Współtworzenie

Wkłady to to, co czyni społeczność open source niesamowitą. Każdy Twój wkład jest **bardzo ceniony**.

### Jak Współtworzyć

1. **Zrób Fork** repozytorium
2. **Utwórz** gałąź funkcji
   ```bash
   git checkout -b feature/niesamowita-funkcja
   ```
3. **Zrób Commit** swoich zmian
   ```bash
   git commit -m "Dodaj niesamowitą funkcję"
   ```
4. **Wypchnij** do gałęzi
   ```bash
   git push origin feature/niesamowita-funkcja
   ```
5. **Otwórz** Pull Request

### Styl Kodu

- W kodzie unikaj nadmiaru komentarzy — dozwolone są tylko istotne komentarze lub te w szczególnie złożonych obszarach
- Używaj klas narzędziowych TailwindCSS do stylowania
- Zachowaj spójność ciemnego motywu
- Utrzymuj JavaScript w wersji vanilla (bez frameworków)
