## Descripción General

**Datvex Prompt LAB** es un repositorio profesional y estructurado que contiene prompts de alta calidad diseñados para tareas especializadas en múltiples sistemas de IA. La plataforma ofrece una interfaz intuitiva con tema oscuro para navegar, buscar y filtrar prompts por tipo, categoría y etiquetas.

Construido pensando en el rendimiento, Datvex ofrece una experiencia ultrarrápida con búsqueda instantánea, scroll infinito y un potente sistema de filtrado — todo sin frameworks pesados.

## Contribuciones

Las contribuciones son lo que hace que la comunidad de código abierto sea increíble. Cualquier contribución que hagas será **muy apreciada**.

### Cómo Contribuir

1. **Haz un Fork** del repositorio
2. **Crea** tu rama de funcionalidad
   ```bash
   git checkout -b feature/funcionalidad-increible
   ```
3. **Haz Commit** de tus cambios
   ```bash
   git commit -m "Añade funcionalidad increíble"
   ```
4. **Haz Push** a la rama
   ```bash
   git push origin feature/funcionalidad-increible
   ```
5. **Abre** un Pull Request

### Estilo de Código

- En el código, evita un exceso de comentarios — solo se permiten comentarios esenciales o en áreas particularmente complejas
- Usa clases de utilidad de TailwindCSS para el estilizado
- Mantén la consistencia del tema oscuro
- Mantén JavaScript vanilla (sin frameworks)
