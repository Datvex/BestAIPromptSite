## Aperçu

**Datvex Prompt LAB** est un répertoire professionnel et structuré contenant des prompts de haute qualité conçus pour des tâches spécialisées sur de multiples systèmes d'IA. La plateforme offre une interface intuitive au thème sombre pour parcourir, rechercher et filtrer les prompts par type, catégorie et tags.

Conçu pour la performance, Datvex offre une expérience ultra-rapide avec une recherche instantanée, un défilement infini et un puissant système de filtrage — le tout sans frameworks lourds.

## Contributions

Les contributions sont ce qui rend la communauté open source incroyable. Toute contribution que vous apportez est **grandement appréciée**.

### Comment Contribuer

1. **Forkez** le répertoire
2. **Créez** votre branche de fonctionnalité
   ```bash
   git checkout -b feature/fonctionnalite-incroyable
   ```
3. **Committez** vos changements
   ```bash
   git commit -m "Ajoute une fonctionnalité incroyable"
   ```
4. **Poussez** vers la branche
   ```bash
   git push origin feature/fonctionnalite-incroyable
   ```
5. **Ouvrez** une Pull Request

### Style de Code

- Dans le code, évitez un excès de commentaires — seuls les commentaires essentiels ou dans les zones particulièrement complexes sont autorisés
- Utilisez les classes utilitaires de TailwindCSS pour le stylisme
- Maintenez la cohérence du thème sombre
- Gardez le JavaScript vanilla (pas de frameworks)
