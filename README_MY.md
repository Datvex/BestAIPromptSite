## Gambaran Keseluruhan

**Datvex Prompt LAB** ialah repositori profesional dan berstruktur yang mengandungi prompt berkualiti tinggi yang direka untuk tugas-tugas khusus merentasi pelbagai sistem AI. Platform ini menyediakan antara muka intuitif bertema gelap untuk menyemak imbas, mencari dan menapis prompt mengikut jenis, kategori dan tag.

Dibina dengan mengutamakan prestasi, Datvex menyampaikan pengalaman sepantas kilat dengan carian serta-merta, tatal tanpa had dan sistem penapisan yang berkuasa — semuanya tanpa rangka kerja yang berat.

## Sumbangan

Sumbangan adalah apa yang menjadikan komuniti sumber terbuka luar biasa. Sebarang sumbangan anda **amat dihargai**.

### Cara Menyumbang

1. **Fork** repositori
2. **Cipta** cawangan ciri anda
   ```bash
   git checkout -b feature/ciri-hebat
   ```
3. **Commit** perubahan anda
   ```bash
   git commit -m "Menambah ciri hebat"
   ```
4. **Push** ke cawangan
   ```bash
   git push origin feature/ciri-hebat
   ```
5. **Buka** Pull Request

### Gaya Kod

- Dalam kod, elakkan lebihan komen — hanya komen penting atau yang berada di kawasan yang sangat kompleks dibenarkan
- Gunakan kelas utiliti TailwindCSS untuk penggayaan
- Kekalkan konsistensi tema gelap
- Kekalkan JavaScript vanilla (tanpa rangka kerja)
