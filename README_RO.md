## Prezentare Generală

**Datvex Prompt LAB** este un depozit profesional și structurat care conține prompturi de înaltă calitate concepute pentru sarcini specializate în mai multe sisteme AI. Platforma oferă o interfață intuitivă cu temă întunecată pentru navigare, căutare și filtrare a prompturilor după tip, categorie și etichete.

Construit cu performanța în minte, Datvex oferă o experiență rapidă ca fulgerul cu căutare instantanee, derulare infinită și un sistem puternic de filtrare — totul fără framework-uri grele.

## Contribuții

Contribuțiile sunt ceea ce face comunitatea open source uimitoare. Orice contribuție este **foarte apreciată**.

### Cum să Contribuiți

1. **Faceți Fork** la depozit
2. **Creați** ramura dvs. de funcționalitate
   ```bash
   git checkout -b feature/functionalitate-incredibila
   ```
3. **Faceți Commit** la modificările dvs.
   ```bash
   git commit -m "Adaugă funcționalitate incredibilă"
   ```
4. **Faceți Push** către ramură
   ```bash
   git push origin feature/functionalitate-incredibila
   ```
5. **Deschideți** un Pull Request

### Stil de Cod

- În cod, evitați un exces de comentarii — sunt permise doar comentariile esențiale sau cele din zone deosebit de complexe
- Folosiți clasele utilitare TailwindCSS pentru stilizare
- Mențineți consistența temei întunecate
- Păstrați JavaScript vanilla (fără framework-uri)
