## Áttekintés

A **Datvex Prompt LAB** egy professzionális, strukturált tárház, amely magas minőségű promptokat tartalmaz, amelyeket több AI rendszer specializált feladataihoz terveztek. A platform intuitív sötét témájú felületet biztosít a promptok böngészéséhez, kereséséhez és szűréséhez típus, kategória és címkék szerint.

A teljesítményre összpontosítva épült, a Datvex villámgyors élményt nyújt azonnali kereséssel, végtelen görgetéssel és hatékony szűrőrendszerrel — mindezt nehéz keretrendszerek nélkül.

## Hozzájárulás

A hozzájárulások teszik a nyílt forráskódú közösséget csodálatossá. Bármilyen hozzájárulásod **nagyon értékelt**.

### Hogyan Járulj Hozzá

1. **Forkold** a tárházat
2. **Hozd létre** a funkció ágazatodat
   ```bash
   git checkout -b feature/csodas-funkcio
   ```
3. **Commitold** a változtatásaidat
   ```bash
   git commit -m "Csodás funkció hozzáadása"
   ```
4. **Pushold** az ágazatba
   ```bash
   git push origin feature/csodas-funkcio
   ```
5. **Nyiss** egy Pull Request-et

### Kódstílus

- A kódban kerüld a kommentek túlzott használatát — csak lényeges kommentek vagy különösen összetett területeken lévők megengedettek
- Használj TailwindCSS utility osztályokat a stílusozáshoz
- Tartsd fenn a sötét téma konzisztenciáját
- Tartsd a JavaScriptet vanillának (nincs keretrendszer)
