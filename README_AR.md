## نظرة عامة

**Datvex Prompt LAB** هو مستودع احترافي ومنظم يحتوي على برومبتات عالية الجودة مصممة لمهام متخصصة عبر أنظمة ذكاء اصطناعي متعددة. توفر المنصة واجهة بديهية بسمة داكنة للتصفح والبحث وتصفية البرومبتات حسب النوع والفئة والوسوم.

مبني مع مراعاة الأداء، يقدم Datvex تجربة سريعة كالبرق مع بحث فوري وتمرير لا نهائي ونظام تصفية قوي — كل ذلك بدون أطر عمل ثقيلة.

## المساهمة

المساهمات هي ما يجعل مجتمع المصادر المفتوحة رائعاً. أي مساهمة تقدمها **موضع تقدير كبير**.

### كيفية المساهمة

1. قم بعمل **Fork** للمستودع
2. **أنشئ** فرع الميزة الخاص بك
   ```bash
   git checkout -b feature/amazing-feature
   ```
3. **قم بتثبيت** تغييراتك
   ```bash
   git commit -m "إضافة ميزة رائعة"
   ```
4. **ادفع** إلى الفرع
   ```bash
   git push origin feature/amazing-feature
   ```
5. **افتح** طلب سحب (Pull Request)

### أسلوب الكود

- في الكود، تجنب الإفراط في التعليقات — يُسمح فقط بالتعليقات الأساسية أو في المناطق المعقدة بشكل خاص
- استخدم فئات أدوات TailwindCSS للتنسيق
- حافظ على اتساق السمة الداكنة
- حافظ على JavaScript الأصلي (بدون أطر عمل)
