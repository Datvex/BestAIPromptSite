## अवलोकन

**Datvex Prompt LAB** एक पेशेवर, संरचित रिपॉजिटरी है जिसमें कई AI सिस्टम में विशेष कार्यों के लिए डिज़ाइन किए गए उच्च-गुणवत्ता वाले प्रॉम्प्ट्स शामिल हैं। प्लेटफ़ॉर्म एक सहज डार्क-थीम इंटरफ़ेस प्रदान करता है जिसमें प्रॉम्प्ट्स को टाइप, श्रेणी और टैग के अनुसार ब्राउज़, खोज और फ़िल्टर किया जा सकता है।

प्रदर्शन को ध्यान में रखकर बनाया गया, Datvex तत्काल खोज, अनंत स्क्रॉल और एक शक्तिशाली फ़िल्टरिंग सिस्टम के साथ बिजली की तेज़ अनुभव प्रदान करता है — सब कुछ भारी फ्रेमवर्क के बिना।

## योगदान

योगदान वही है जो ओपन सोर्स समुदाय को अद्भुत बनाता है। आपका कोई भी योगदान **बहुत सराहा जाएगा**।

### कैसे योगदान करें

1. रिपॉजिटरी को **Fork** करें
2. अपनी फ़ीचर ब्रांच **बनाएँ**
   ```bash
   git checkout -b feature/amazing-feature
   ```
3. अपने बदलाव **कमिट** करें
   ```bash
   git commit -m "अद्भुत फ़ीचर जोड़ें"
   ```
4. ब्रांच में **पुश** करें
   ```bash
   git push origin feature/amazing-feature
   ```
5. एक Pull Request **खोलें**

### कोड शैली

- कोड में, टिप्पणियों की अधिकता से बचें — केवल आवश्यक टिप्पणियाँ या विशेष रूप से जटिल क्षेत्रों में ही अनुमति है
- स्टाइलिंग के लिए TailwindCSS यूटिलिटी क्लासेज का उपयोग करें
- डार्क थीम की संगति बनाए रखें
- JavaScript वेनिला रखें (कोई फ्रेमवर्क नहीं)
