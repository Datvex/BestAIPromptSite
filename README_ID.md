## Ikhtisar

**Datvex Prompt LAB** adalah repositori profesional dan terstruktur yang berisi prompt berkualitas tinggi yang dirancang untuk tugas-tugas khusus di berbagai sistem AI. Platform ini menyediakan antarmuka intuitif bertema gelap untuk menelusuri, mencari, dan memfilter prompt berdasarkan jenis, kategori, dan tag.

Dibangun dengan mempertimbangkan performa, Datvex memberikan pengalaman secepat kilat dengan pencarian instan, scroll tak terbatas, dan sistem pemfilteran yang kuat — semuanya tanpa framework berat.

## Kontribusi

Kontribusi adalah yang membuat komunitas open source luar biasa. Setiap kontribusi Anda **sangat dihargai**.

### Cara Berkontribusi

1. **Fork** repositori
2. **Buat** branch fitur Anda
   ```bash
   git checkout -b feature/fitur-luar-biasa
   ```
3. **Commit** perubahan Anda
   ```bash
   git commit -m "Menambahkan fitur luar biasa"
   ```
4. **Push** ke branch
   ```bash
   git push origin feature/fitur-luar-biasa
   ```
5. **Buka** Pull Request

### Gaya Kode

- Dalam kode, hindari kelebihan komentar — hanya komentar penting atau yang ada di area yang sangat kompleks yang diizinkan
- Gunakan kelas utilitas TailwindCSS untuk styling
- Pertahankan konsistensi tema gelap
- Pertahankan JavaScript vanilla (tanpa framework)
