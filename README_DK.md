## Oversigt

**Datvex Prompt LAB** er et professionelt, struktureret repository indeholdende højkvalitets-prompts designet til specialiserede opgaver på tværs af flere AI-systemer. Platformen tilbyder en intuitiv mørkt-tema grænseflade til at browse, søge og filtrere prompts efter type, kategori og tags.

Bygget med ydeevne i tankerne, leverer Datvex en lynhurtig oplevelse med øjeblikkelig søgning, uendelig scroll og et kraftfuldt filtreringssystem — alt sammen uden tunge frameworks.

## Bidrag

Bidrag er det, der gør open source-fællesskabet fantastisk. Ethvert bidrag du yder, er **meget værdsat**.

### Hvordan Man Bidrager

1. **Fork** repositoriet
2. **Opret** din feature-gren
   ```bash
   git checkout -b feature/fantastisk-funktion
   ```
3. **Commit** dine ændringer
   ```bash
   git commit -m "Tilføj fantastisk funktion"
   ```
4. **Push** til grenen
   ```bash
   git push origin feature/fantastisk-funktion
   ```
5. **Åbn** en Pull Request

### Kodestil

- I koden, undgå et overskud af kommentarer — kun væsentlige kommentarer eller dem i særligt komplekse områder er tilladt
- Brug TailwindCSS utility-klasser til styling
- Bevar det mørke temas konsistens
- Hold JavaScript vanilla (ingen frameworks)
