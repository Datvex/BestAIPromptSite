## Přehled

**Datvex Prompt LAB** je profesionální, strukturovaný repozitář obsahující vysoce kvalitní prompty navržené pro specializované úlohy v různých AI systémech. Platforma poskytuje intuitivní rozhraní s tmavým motivem pro procházení, vyhledávání a filtrování promptů podle typu, kategorie a štítků.

Postavený s důrazem na výkon, Datvex nabízí bleskově rychlý zážitek s okamžitým vyhledáváním, nekonečným posouváním a výkonným systémem filtrování — vše bez těžkých frameworků.

## Přispívání

Příspěvky jsou to, co dělá komunitu open source úžasnou. Jakýkoli váš příspěvek je **velmi ceněný**.

### Jak Přispět

1. **Forkněte** repozitář
2. **Vytvořte** vaši feature větev
   ```bash
   git checkout -b feature/uzasna-funkce
   ```
3. **Commitněte** vaše změny
   ```bash
   git commit -m "Přidat úžasnou funkci"
   ```
4. **Pushněte** do větve
   ```bash
   git push origin feature/uzasna-funkce
   ```
5. **Otevřete** Pull Request

### Styl Kódu

- V kódu se vyhněte nadbytku komentářů — povoleny jsou pouze podstatné komentáře nebo komentáře ve zvláště složitých oblastech
- Používejte TailwindCSS utility třídy pro stylování
- Udržujte konzistentnost tmavého motivu
- Ponechte JavaScript vanilla (žádné frameworky)
