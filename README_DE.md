## Überblick

**Datvex Prompt LAB** ist ein professionelles, strukturiertes Repository mit hochwertigen Prompts, die für spezialisierte Aufgaben in verschiedenen KI-Systemen entwickelt wurden. Die Plattform bietet eine intuitive, dunkel gestaltete Oberfläche zum Durchsuchen, Suchen und Filtern von Prompts nach Typ, Kategorie und Tags.

Mit Fokus auf Leistung gebaut, bietet Datvex ein blitzschnelles Erlebnis mit Sofortsuche, unendlichem Scrollen und einem leistungsstarken Filtersystem — alles ohne schwere Frameworks.

## Mitwirken

Beiträge machen die Open-Source-Community zu einem großartigen Ort. Jeder Beitrag wird **sehr geschätzt**.

### Wie man Beiträgt

1. **Forken** Sie das Repository
2. **Erstellen** Sie Ihren Feature-Branch
   ```bash
   git checkout -b feature/grossartige-funktion
   ```
3. **Committen** Sie Ihre Änderungen
   ```bash
   git commit -m "Fügt großartige Funktion hinzu"
   ```
4. **Pushen** Sie zum Branch
   ```bash
   git push origin feature/grossartige-funktion
   ```
5. **Öffnen** Sie einen Pull Request

### Code-Stil

- Im Code einen Überschuss an Kommentaren vermeiden — nur wesentliche Kommentare oder solche in besonders komplexen Bereichen sind erlaubt
- TailwindCSS Utility-Klassen für das Styling verwenden
- Die Konsistenz des dunklen Themes beibehalten
- JavaScript vanilla halten (keine Frameworks)
