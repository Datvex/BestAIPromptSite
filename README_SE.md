## Översikt

**Datvex Prompt LAB** är ett professionellt, strukturerat förråd som innehåller högkvalitativa prompter designade för specialiserade uppgifter i flera AI-system. Plattformen erbjuder ett intuitivt gränssnitt med mörkt tema för att bläddra, söka och filtrera prompter efter typ, kategori och taggar.

Byggt med prestanda i åtanke, levererar Datvex en blixtsnabb upplevelse med omedelsök, oändlig scrollning och ett kraftfullt filtreringssystem — allt utan tunga ramverk.

## Bidra

Bidrag är det som gör öppen källkods-communityn fantastisk. Alla bidrag du gör är **mycket uppskattade**.

### Hur man Bidrar

1. **Forka** förrådet
2. **Skapa** din funktionsgren
   ```bash
   git checkout -b feature/fantastisk-funktion
   ```
3. **Committa** dina ändringar
   ```bash
   git commit -m "Lägg till fantastisk funktion"
   ```
4. **Pusha** till grenen
   ```bash
   git push origin feature/fantastisk-funktion
   ```
5. **Öppna** en Pull Request

### Kodstil

- I koden, undvik överflödiga kommentarer — bara väsentliga kommentarer eller de i särskilt komplexa områden är tillåtna
- Använd TailwindCSS utility-klasser för styling
- Bibehåll det mörka temats konsekvens
- Håll JavaScript vanilla (inga ramverk)
