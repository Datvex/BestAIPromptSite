## Обзор

**Datvex Prompt LAB** — это профессиональный структурированный репозиторий, содержащий высококачественные промпты, разработанные для специализированных задач в различных системах ИИ. Платформа предоставляет интуитивный интерфейс с тёмной темой для просмотра, поиска и фильтрации промптов по типу, категории и тегам.

Созданный с фокусом на производительность, Datvex обеспечивает молниеносный опыт использования с мгновенным поиском, бесконечной прокруткой и мощной системой фильтрации — всё без тяжёлых фреймворков.

## Вклад в Проект

Вклады — это то, что делает сообщество открытого исходного кода замечательным. Любой ваш вклад **очень ценится**.

### Как Внести Вклад

1. **Сделайте Fork** репозитория
2. **Создайте** ветку для вашей функции
   ```bash
   git checkout -b feature/потрясающая-функция
   ```
3. **Зафиксируйте** ваши изменения
   ```bash
   git commit -m "Добавляет потрясающую функцию"
   ```
4. **Отправьте** в ветку
   ```bash
   git push origin feature/потрясающая-функция
   ```
5. **Откройте** Pull Request

### Стиль Кода

- В коде избегайте избытка комментариев — допускаются только существенные комментарии или комментарии в особенно сложных местах
- Используйте утилитарные классы TailwindCSS для стилизации
- Поддерживайте единообразие тёмной темы
- Используйте чистый JavaScript (без фреймворков)
