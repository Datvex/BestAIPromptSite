## Visão Geral

**Datvex Prompt LAB** é um repositório profissional e estruturado contendo prompts de alta qualidade projetados para tarefas especializadas em vários sistemas de IA. A plataforma fornece uma interface intuitiva com tema escuro para navegar, pesquisar e filtrar prompts por tipo, categoria e tags.

Construído com foco em desempenho, o Datvex oferece uma experiência extremamente rápida com pesquisa instantânea, rolagem infinita e um poderoso sistema de filtros — tudo isso sem frameworks pesados.

## Contribuindo

Contribuições são o que tornam a comunidade open source incrível. Qualquer contribuição que você fizer é **muito apreciada**.

### Como Contribuir

1. **Faça um Fork** do repositório
2. **Crie** sua branch de funcionalidade
   ```bash
   git checkout -b feature/nova-funcionalidade
   ```
3. **Faça commit** das suas alterações
   ```bash
   git commit -m "Adicionar nova funcionalidade"
   ```
4. **Envie** para a branch
   ```bash
   git push origin feature/nova-funcionalidade
   ```
5. **Abra** um Pull Request

### Estilo de Código

- No código, evite excesso de comentários — apenas comentários essenciais ou aqueles em áreas particularmente complexas são permitidos
- Use classes utilitárias do TailwindCSS para estilização
- Mantenha a consistência do tema escuro
- Mantenha JavaScript puro (sem frameworks)
