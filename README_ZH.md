## 概述

**Datvex Prompt LAB** 是一个专业的结构化仓库，包含为多个 AI 系统的专业任务设计的高质量提示词。该平台提供直观的深色主题界面，用于浏览、搜索和按类型、分类和标签过滤提示词。

以性能为核心构建，Datvex 提供闪电般快速的体验，具有即时搜索、无限滚动和强大的过滤系统 ——一切都不依赖重型框架。

## 贡献

贡献是使开源社区如此美好的原因。您的任何贡献都将**深受感激**。

### 如何贡献

1. **Fork** 仓库
2. **创建**您的功能分支
   ```bash
   git checkout -b feature/amazing-feature
   ```
3. **提交**您的更改
   ```bash
   git commit -m "添加了令人惊叹的功能"
   ```
4. **推送**到分支
   ```bash
   git push origin feature/amazing-feature
   ```
5. **打开** Pull Request

### 代码风格

- 在代码中，避免过多的注释 ——仅允许必要的注释或在特别复杂的区域添加注释
- 使用 TailwindCSS 实用类进行样式设计
- 保持深色主题的一致性
- 保持原生 JavaScript（不使用框架）
