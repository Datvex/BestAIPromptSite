## Prehľad

**Datvex Prompt LAB** je profesionálny, štruktúrovaný repozitár obsahujúci vysokokvalitné prompty navrhnuté pre špecializované úlohy v rôznych AI systémoch. Platforma poskytuje intuitívne rozhranie s tmavou témou na prehliadanie, vyhľadávanie a filtrovanie promptov podľa typu, kategórie a značiek.

Postavený s dôrazom na výkon, Datvex ponúka bleskurýchly zážitok s okamžitým vyhľadávaním, nekonečným posúvaním a výkonným systémom filtrovania — všetko bez ťažkých frameworkov.

## Prispievanie

Príspevky sú to, čo robí komunitu open source úžasnou. Akýkoľvek váš príspevok je **veľmi cenený**.

### Ako Prispieť

1. **Forkujte** repozitár
2. **Vytvorte** vašu feature vetvu
   ```bash
   git checkout -b feature/uzasna-funkcia
   ```
3. **Commitnite** vaše zmeny
   ```bash
   git commit -m "Pridať úžasnú funkciu"
   ```
4. **Pushnite** do vetvy
   ```bash
   git push origin feature/uzasna-funkcia
   ```
5. **Otvorte** Pull Request

### Štýl Kódu

- V kóde sa vyhnite nadbytku komentárov — povolené sú len podstatné komentáre alebo komentáre v obzvlášť zložitých oblastiach
- Používajte TailwindCSS utility triedy na štylizovanie
- Udržujte konzistentnosť tmavej témy
- Ponechajte JavaScript vanilla (žiadne frameworky)
