## Yleiskatsaus

**Datvex Prompt LAB** on ammattimainen, jäsennelty kokoelma, joka sisältää laadukkaita prompteja, jotka on suunniteltu erikoistehtäviin useissa tekoälyjärjestelmissä. Alusta tarjoaa intuitiivisen tumman teeman käyttöliittymän promptien selaamiseen, hakemiseen ja suodattamiseen tyypin, kategorian ja tunnisteiden mukaan.

Suorituskyky mielessä rakennettu Datvex tarjoaa salamannopean kokemuksen välittömällä haulla, loputtomalla vierityksellä ja tehokkaalla suodatusjärjestelmällä — kaikki ilman raskaita ohjelmistokehyksiä.

## Osallistuminen

Osallistumiset tekevät avoimen lähdekoodin yhteisöstä hämmästyttävän. Kaikki osallistumisesi ovat **suuresti arvostettuja**.

### Kuinka Osallistua

1. **Forkkaa** kokoelma
2. **Luo** ominaisuushaarasi
   ```bash
   git checkout -b feature/mahtava-ominaisuus
   ```
3. **Committaa** muutoksesi
   ```bash
   git commit -m "Lisää mahtava ominaisuus"
   ```
4. **Pushaa** haaraan
   ```bash
   git push origin feature/mahtava-ominaisuus
   ```
5. **Avaa** Pull Request

### Koodityyli

- Koodissa vältä liiallisia kommentteja — vain olennaiset kommentit tai erityisen monimutkaisten alueiden kommentit ovat sallittuja
- Käytä TailwindCSS utility-luokkia tyylittelyyn
- Ylläpidä tumman teeman johdonmukaisuutta
- Pidä JavaScript vaniljana (ei ohjelmistokehyksiä)
