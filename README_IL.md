## סקירה כללית

**Datvex Prompt LAB** הוא מאגר מקצועי ומובנה המכיל פרומפטים איכותיים שתוכננו למשימות מתמחות במערכות בינה מלאכותית מרובות. הפלטפורמה מספקת ממשק אינטואיטיבי בערכת נושא כהה לגלישה, חיפוש וסינון פרומפטים לפי סוג, קטגוריה ותגיות.

בנוי עם ביצועים בראש סדר העדיפויות, Datvex מספק חוויה מהירה כברק עם חיפוש מיידי, גלילה אינסופית ומערכת סינון רבת עוצמה — הכל ללא פריימוורקים כבדים.

## תרומה

תרומות הן מה שהופך את קהילת הקוד הפתוח למדהימה. כל תרומה שלך **מוערכת מאוד**.

### איך לתרום

1. **עשה Fork** למאגר
2. **צור** את ענף התכונה שלך
   ```bash
   git checkout -b feature/amazing-feature
   ```
3. **בצע Commit** לשינויים שלך
   ```bash
   git commit -m "הוספת תכונה מדהימה"
   ```
4. **עשה Push** לענף
   ```bash
   git push origin feature/amazing-feature
   ```
5. **פתח** Pull Request

### סגנון קוד

- בקוד, הימנע מעודף הערות — רק הערות חיוניות או כאלה באזורים מורכבים במיוחד מותרות
- השתמש במחלקות utility של TailwindCSS לעיצוב
- שמור על עקביות ערכת הנושא הכהה
- שמור על JavaScript וניל (ללא פריימוורקים)
